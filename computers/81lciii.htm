Installing OS 8 & OS 8.1 on 68030 Macs
Thanks to Vintage Box : 68k Mac - OS 8/030 for the OS 8 & 8.1 hacks.
Description:
Following are methods for installing OS 8.0 and 8.1 on 68030 Macs made with 32 bit clean ROMs.
These Macs have been reported successful with both OS 8.0 and 8.1.
This page shows how to do it with the SE/30.
Hardware Requirements:
 • Macintosh IIci, IIfx, IIsi, IIvi, IIvx, LC III, LC520, 550, 560; Performa 275, 450, 460-467, 600;
   Color Classic II; PowerBook 150, 160, 165, 165c, 180, 180c; PowerBook Duo 210, 230, 250, 270c.
 • Minimum of 12MB of RAM.
 • Single hard drive, minimum of 44MB free space.
 • OS 7.0.1 minimum.
 • CD ROM drive.
OS 8.0 Install:
  1. Hard drive must have boot OS 7.0.1 (or higher), and also ResEdit 2.1.3, System_Picker 1.1a3, os8-030patch.rsrc, and Wish I Were 2.0.
  2. CD ROM drive to have OS 8.0 or OS 8.1 CD.
  3. Boot from hard drive OS 7.0.1 (or higher).
  4. Drag Wish I Were control panel and Wish I Were extension to hard drive OS 7.x.x System Folder icon.
  5. Set Wish I Were control panel to 'Quadra 700', close all control panels, and reboot.
  6. Launch the 8.0 Installer and select Installer options "Clean Install" and "Easy Install".
    NOTE: On the Mac OS 8.1 CD, the OS 8.0 Installer is located at
    Full Install Pieces-->Software Installers-->System Software-->Mac OS 8-->.
  7. When installer is done restart with Network Access Disk 7.5 floppy and do ResEdit hacks:
  8. ResEdit hacks to 8.0 System file:
     • Replace 'boot' resource 'ID 2' with same resource from 'os8-030patch.rsrc'.
     • Replace 'gbly' resource 'ID -16385' with same resource from 'os8-030patch.rsrc'.
     • 'gusd' resource 'ID 1' at offset '000010', replace data '0016 0006' with data
       '00?? 0005', where ?? is the hexidecimal code as shown in this pull-down menu:
    .
     • Quit ResEdit.
  9. Trash the 'Monitors and Sound' control panel.
  10. From the Apple Extras folder drag both the 'Monitors' and 'Sound' control panels to the System Folder icon.
  11. ResEdit hack to 8.0 Finder file: (optional)
     • 'SIZE' resource 'ID -1', change 'Size' to '1310720'.
     • Quit ResEdit.
  12. Use System Picker to select hard drive OS 8.0 for reboot.
OS 8.1 Update Install:
  1. Hard drive must have OS 8.0 which was installed per above, and also ResEdit 2.1.3, System_Picker 1.1a3, os8-030patch.rsrc, 8.1 Update, and Wish I Were 2.0.
  2. Boot from hard drive OS 8.0.
  3. Drag Wish I Were control panel and Wish I Were extension to OS 8.0 System folder icon.
  4. Set Extensions Manager control panel to 'My Settings' and check all boxes.
  5. Set Wish I Were control panel to 'Quadra 700'. Close all control panels.
  6. Use System Picker to reboot from hard drive OS 8.0.
  7. Open the 'Update to Mac OS 8.1' installer and in Options uncheck 'Update Apple Hard Drive Drivers'.
  8. Run 'Update to Mac OS 8.1' installer.
  9. When installer is done restart with Network Access Disk 7.5 floppy and do ResEdit hacks:
  10. ResEdit hacks to 8.1 System file:
     • Replace 'gbly' resource 'ID -16385' with the same resource from 'os8-030patch.rsrc'.
     • 'gusd' resource 'ID 1' at offset '000010', replace data '0016 0006' with data
       '00?? 0006', where ?? is the same code that was used for installing OS 8.0.
  11. ResEdit hack to 8.1 Finder file: (optional)
     • 'SIZE' resource 'ID -1', change 'Size' to '1310720'.
  12. Quit ResEdit.
  13. Remove Wish I Were control panel and extension. (optional)
  14. Use System Picker to select hard drive OS 8.1 for reboot.
Both OS 8.0 and 8.1 have been successful with these Macs:
 • IIci  (20/250, built-in video)
 • IIci  (64/1000, Spectrum/24 PDQ+, DayStar Turbo 040 40MHz)
 • IIfx  (32/250, Macintosh Display Card 670 8•24 820-0600-A)
 • IIsi  (17/250, built-in video)
 • LCIII (36/80) (field report, OS 8.0)
